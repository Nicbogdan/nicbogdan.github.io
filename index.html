Build and run container d1xdemo

This will build a docker image containing a X window environment in which I run Descent 1 demo. The d1x-rebirth file comes from dxx-rebirth project which allows to run both Descent 1 and 2 games for diferent platforms.
The game can be run very similarly like on pc, the advandage here is that you can interact with the game from the browser.
To run the container from a Linux machine that has docker installed follow the steps below:
  1. Git clone https://github.com/Nicbogdan/dbuild1.git

  2. Run from terminal cd dbuild1

  3. Run from terminal unzip d1x-rebirth.zip

  4. Run from terminal docker build -t d1xdemo .

  5. Run from terminal docker run -p 6080:6080 -p 42424:42424 --name d1xgame d1xdemo

After this in terminal it shows a vnc link.
Open the link in a browser and you will get a graphical interface inside the container, that has a terminal running.
From that terminal run ./d1x-rebirth and the game starts.
Video demos coming soon.


To clean up :
  • Run from terminal docker stop d1xgame

  • Run from terminal docker rm d1xgame

  • Run from terminal docker rmi d1xdemo
