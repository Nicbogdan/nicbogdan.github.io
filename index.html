Build and run container d1xdemo

This will build a docker image containing a X window environment in which I run Descent 1 demo
The gist of Dockerfile is located at here
Video demo will be added soon.
To run the container follow the steps below:
  1. Git clone https://github.com/Nicbogdan/dbuild1.git

  2. Unzip locally d1x-rebirth.zip

  3. Run from terminal docker -t d1xdemo .

  4. Run from terminal docker run -p 6080:6080 -p 42424:42424 --name d1xgame d1xdemo

To stop container :
  • Run from terminal docker stop d1xgame

To start again :
  • Run from terminal docker start d1xgame

To delete the container :
  • Run from terminal docker rm d1xgame

If you want to delete the image :
  • Run from terminal docker rmi -f d1xdemo

To recreate the container and the image run :
  • Run from terminal docker run -p 6080:6080 -p 42424:42424 --name d1xgame d1xdemo
